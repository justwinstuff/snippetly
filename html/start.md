# Basic HTML Boilerplate
A simple boilerplate to start any HTML page.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Website</title>
</head>
<body>
    <h1>Snippetly</h1>
</body>
</html>
```
